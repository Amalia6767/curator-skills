# 交互组交付技能 (Interaction Team Deliverables)

## 📋 技能简介

本技能用于展厅/展馆类投标项目中，将**策划组**完成的策划方案转化为**交互组**可执行的设计交付文件，确保满足招标文件的技术评分要求。

## 🎯 适用场景

- 科技展厅设计施工一体化项目
- 企业展厅/体验中心投标
- 需要策划组与交互组协作的展陈项目
- 涉及多风格切换、AI交互、内容管理平台的展厅项目

```
interaction-team-deliverables/
```

## 🚀 使用方法

### 1. 阅读技能文档

```
查看 SKILL.md 了解完整的交付指南
```

### 2. 根据项目情况定制

SKILL.md 中的内容基于通用评分项设计，使用时需根据具体项目的招标文件进行调整：

| 需定制项 | 说明 |
|---------|------|
| 评分项对照表 | 根据实际招标文件的技术评分表更新 |
| 风格定义 | 根据甲方实际的多客群需求更新 |
| AI交互场景 | 根据甲方业务能力和展示需求调整 |
| 策划篇索引 | 根据实际策划文档的章节结构更新 |

### 3. 召开策划-交互会议

使用 SKILL.md 作为会议议程指南，确保：
- 策划组传达核心设计理念
- 交互组理解评分项要求
- 明确各阶段交付物内容

## 📊 核心内容概览

| 模块 | 内容 |
|------|------|
| 评分项对照 | 交互相关的技术评分项及分值 |
| 多风格切换 | A/B/C/D/E等风格的定义和切换维度 |
| 交互场景 | AI交互场景的设计要点和技术实现 |
| 内容平台 | 内容设计平台的功能需求 |
| 数据可视化 | 信息分层和可视化规范 |
| 交付清单 | 投标阶段和深化阶段的交付物清单 |
| 协作流程 | 策划组与交互组的协作流程 |
