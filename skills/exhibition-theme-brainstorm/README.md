# 展陈主题头脑风暴 Skill

本 skill 用于系统化推导展陈主题，通过"九层递进法"从宏观理念、企业特点、价值传递等维度，最终形成有哲学深度、有企业特色、可视觉化的展陈主题。

## 文件结构

```
exhibition-theme-brainstorm/
├── SKILL.md          # 核心技能说明与工作流程
└── README.md         # 本文件
```

## 输入

### 必需输入
- **企业/机构基本信息**：企业名称、行业、核心业务、使命愿景、核心价值观
- **项目背景**：展厅定位、目标受众、空间特征、面积限制

### 可选输入（强烈推荐）
- 企业文化、发展历程
- 核心产品/服务
- 行业地位、差异化优势
- 目标受众画像
- 创始人/领导人理念

## 输出

生成 `展陈主题推导报告.md` 文件，包含：

1. **信息收集**：企业信息、项目背景
2. **九层递进推导**：
   - 第一层：宏观理念（哲学基础）
   - 第二层：企业特点（核心逻辑）
   - 第三层：创始人/领导人理念
   - 第四层：使命愿景
   - 第五层：核心价值观
   - 第六层：价值传递路径
   - 第七层：空间命名
   - 第八层：展陈主题（最终输出）
   - 第九层：设计元素提取
3. **主题解读**：核心解读、关键词提取、设计元素建议

## 使用场景

- **项目启动阶段**：需要确定展陈主题和核心理念
- **方案深化阶段**：需要将主题转化为设计元素和空间命名
- **创意头脑风暴**：需要系统化方法激发创意

## 工作流程

1. **信息收集**：收集企业信息、项目背景
2. **九层递进推导**：按照框架逐层推导
3. **主题验证**：验证主题是否满足要求
4. **主题深化**：形成主题解读和设计元素建议

## 核心方法：九层递进法

从宏观理念出发，逐层递进到最终主题：
- **哲学层**：宏观理念、创始人理念
- **企业层**：企业特点、使命愿景、核心价值观
- **价值层**：价值传递路径
- **空间层**：空间命名
- **主题层**：展陈主题
- **视觉层**：设计元素提取

## 优秀案例

- **某智慧能源集团**："无界体验 无限价值——数智能源 开启美好生活"
  - 从《老子·道德经》出发，结合需-供逻辑，形成无界厅→无限厅的空间命名
  
- **某消费金融公司**："棱镜3.0——液态叙事的数智展厅"
  - 从光学原理出发，结合AI驱动的普惠金融基座，形成棱镜理念

## 与其他 skill 的关系

- `proposal-design`：使用主题推导结果，形成完整方案
- `ppt-layout`：将主题转化为PPT视觉表达
- 两者配合使用，完成从主题到方案的完整流程
