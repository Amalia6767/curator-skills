# 展陈主题头脑风暴

## 目标

系统化推导展陈主题，确保主题：
- **有哲学深度**：从宏观理念出发，有文化底蕴
- **有企业特色**：紧密结合企业/机构的核心特点
- **有视觉表达**：可转化为设计元素和空间命名
- **有价值传递**：清晰的价值传递路径

## 输入

### 必需信息
1. **企业/机构基本信息**
   - 企业名称、行业、核心业务
   - 使命愿景、核心价值观
   - 创始人/领导人理念（如有）

2. **项目背景**
   - 展厅定位、目标受众
   - 空间特征、面积限制
   - 核心展示内容

### 可选信息（强烈推荐）
- 企业文化、发展历程
- 核心产品/服务
- 行业地位、差异化优势
- 目标受众画像

## 主题推导框架：九层递进法

### 第一层：宏观理念（哲学基础）

**目标**：找到可以支撑主题的哲学思想或文化理念

**方法**：
- 从中国传统文化中寻找（如：道生一、一生二、二生三、三生万物）
- 从西方哲学中寻找（如：存在主义、实用主义）
- 从行业经典理论中寻找（如：动态能力理论）
- 从自然现象中寻找（如：棱镜、光谱、液态）

**输出示例**：
```
宏观理念：《老子·道德经》第四十二章
- 道生一：从无到有
- 一生二：一分为二，即阴阳
- 二生三：阴阳相冲，而后阴盛阳弱，或阴弱阳盛，或阴阳中和
- 三生万物：或阴、或阳、或和，三种趋势相互转化孕育万物
```

### 第二层：企业特点（核心逻辑）

**目标**：提炼企业/机构的核心业务逻辑和特点

**方法**：
- 分析业务模式（如：需-供、角色-场景-智能）
- 分析核心能力（如：万物互联、数智化）
- 分析组织特点（如：网络型组织、网络型业务）

**输出示例**：
```
企业特点：
- 需-供：从用户需求出发
- 角色-场景-智能：三大核心要素
- 万物互联：能量全价值链开发利用
```

### 第三层：创始人/领导人理念

**目标**：融入创始人或领导人的核心思想

**方法**：
- 提取核心理念（如：以人为本、无为而治）
- 提取价值主张（如：源自客户、成就彼此、共创生态）

**输出示例**：
```
创始人理念：
- 以人为本，无为而治
- 企业之道：源自客户、成就彼此、共创生态
```

### 第四层：使命愿景

**目标**：将使命愿景转化为主题表达

**方法**：
- 提取关键词（如：创建现代能源体系、提高人民生活品质）
- 转化为价值主张（如：数智能源、开启美好生活）

**输出示例**：
```
使命愿景：
- 创建现代能源体系，提高人民生活品质
- 成为一家受人尊敬的创新型智慧企业
→ 转化为：数智能源 开启美好生活
```

### 第五层：核心价值观

**目标**：将价值观融入主题表达

**方法**：
- 列出核心价值观（如：执守良知、崇尚自驱、利他共生、成在数智）
- 提取核心关键词

**输出示例**：
```
核心价值观：
- 执守良知、崇尚自驱、利他共生、成在数智
→ 核心关键词：良知、自驱、共生、数智
```

### 第六层：价值传递路径

**目标**：定义从企业到用户的价值传递逻辑

**方法**：
- 企业视角：传递价值 → 体验价值 → 感知价值 → 利用价值 → 重构价值 → 创造价值
- 用户视角：发现需求 → 解决问题 → 用户反馈

**输出示例**：
```
价值传递路径：
企业视角：传递价值 → 体验价值 → 感知价值 → 利用价值 → 重构价值 → 创造价值
用户视角：发现需求 → 解决问题 → 用户反馈
```

### 第七层：空间命名

**目标**：将主题转化为空间命名

**方法**：
- 从主题中提取关键词
- 形成对仗或呼应的空间名称
- 确保名称有寓意且易理解

**输出示例**：
```
空间命名：
- 无界厅（发现需求）→ 无限厅（解决问题）
- 从"无界"到"无限"，体现价值创造的无限可能
```

### 第八层：展陈主题（最终输出）

**目标**：形成完整的主题表达

**方法**：
- 组合关键词形成主题
- 确保主题简洁有力、朗朗上口
- 可包含副标题补充说明

**输出示例**：
```
展陈主题：
无界体验 无限价值
——数智能源 开启美好生活

核心解读：
- 无界体验：打破边界，发现需求
- 无限价值：解决问题，创造价值
- 数智能源：企业定位
- 开启美好生活：价值目标
```

### 第九层：设计元素提取

**目标**：将主题转化为可视觉化的设计元素

**方法**：
- 从主题中提取视觉符号（如：网络、对称、圆环）
- 从理念中提取视觉隐喻（如：棱镜、光谱、液态）
- 确保元素有文化内涵和视觉美感

**输出示例**：
```
设计元素：
- 网络：互联-协调（生命观：经络、脉络）
- 对称：平衡-秩序（生命律：熵增熵减、内外平衡）
- 圆环：从无到有-永恒（生命力：身心健康）

空间应用：
- 中庭"方圆之间"：天圆地方，阴阳平衡
- 方正的浮雕：秩序、原则
- 圆融的装置：智圆、融通
```

## 工作流程

### 步骤1：信息收集
1. 收集企业/机构基本信息
2. 收集项目背景信息
3. 收集可选信息（文化、产品、受众等）

### 步骤2：九层递进推导
按照九层框架，逐层推导：
1. 宏观理念 → 2. 企业特点 → 3. 创始人理念 → 4. 使命愿景 → 5. 核心价值观 → 6. 价值传递路径 → 7. 空间命名 → 8. 展陈主题 → 9. 设计元素

### 步骤3：主题验证
验证主题是否满足：
- ✅ 有哲学深度
- ✅ 有企业特色
- ✅ 有视觉表达
- ✅ 有价值传递
- ✅ 简洁有力、朗朗上口

### 步骤4：主题深化
- 形成主题解读文档
- 提取核心关键词
- 设计元素可视化建议

## 优秀案例参考

### 案例1：某智慧能源集团
- **宏观理念**：《老子·道德经》"道生一，一生二，二生三，三生万物"
- **企业特点**：需-供、角色-场景-智能、万物互联
- **主题**："无界体验 无限价值——数智能源 开启美好生活"
- **空间命名**：无界厅（发现需求）→ 无限厅（解决问题）
- **设计元素**：网络、对称、圆环

### 案例2：某消费金融公司（棱镜理念）
- **宏观理念**：光学原理（棱镜折射）
- **企业特点**：AI驱动的普惠金融基座
- **主题**：棱镜3.0——液态叙事的数智展厅
- **空间命名**：入口区→过渡区→业务区→科技区→体验区→出口区
- **设计元素**：棱镜、光谱、数据星河

## 注意事项

1. **避免泛化**：主题要有具体指向，避免"高端大气上档次"等空泛表达
2. **确保一致性**：九层推导要逻辑一致，不能前后矛盾
3. **可视觉化**：主题要能转化为设计元素，不能只是文字游戏
4. **有文化底蕴**：从哲学/文化中寻找支撑，提升主题深度
5. **朗朗上口**：主题要简洁有力，易于记忆和传播

## 常见问题

### Q1：找不到合适的宏观理念怎么办？
**A**：可以从以下方向寻找：
- 行业经典理论（如：动态能力理论、价值链理论）
- 自然现象（如：棱镜、光谱、液态、固态）
- 文化符号（如：天圆地方、阴阳平衡）
- 企业自身的发展历程（如：从无到有、从小到大）

### Q2：如何确保主题有企业特色？
**A**：确保主题中：
- 包含企业的核心业务逻辑关键词
- 体现企业的差异化优势
- 呼应企业的使命愿景

### Q3：如何将主题转化为设计元素？
**A**：
- 从主题中提取视觉符号（如：网络、圆环、棱镜）
- 从理念中提取视觉隐喻（如：无界→无限、固态→液态）
- 确保元素有文化内涵和视觉美感
