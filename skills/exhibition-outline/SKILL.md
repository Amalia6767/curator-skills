# 展陈大纲生成

## 目标

根据策划方案生成展陈大纲，以3页PPT格式呈现：
- **第一页**：展陈主题（核心主题、副标题、理念解读）
- **第二页**：诠释与过渡（主题诠释、叙事逻辑、空间规划）
- **第三页**：详细的展陈大纲（要点版）（分区、内容、展示形式）

确保：
- **结构清晰**：三页内容层次分明，逻辑递进
- **要点突出**：每页聚焦核心信息，避免冗余
- **便于执行**：可直接用于PPT制作或方案汇报

## 输入

- **策划方案**（Markdown格式）：通常来自 `proposal-design` skill 的输出
  - 例如：`投标方案_策划篇.md`
- **可选**：展陈主题推导报告（用于理解主题的推导逻辑）

## 输出格式

生成 `展陈大纲_PPT版.md` 文件，包含3页PPT内容：

### 第一页：展陈主题

```markdown
## 第一页：展陈主题

### 主标题
[展陈主题名称]

### 副标题
[副标题或slogan]

### 主题核心解读
- **一句话概括**：[用一句话概括主题的核心内涵]
- **空间维度**：[主题在空间/业务维度的含义]
- **时间维度**：[主题在时间/价值维度的含义]
- **统一理念**：[空间与时间的统一，形成完整理念]

### 核心理念
[1-2句话说明核心理念，如棱镜理念、液态叙事等]

### 视觉元素建议
- [主视觉建议，如：数据星河、棱镜折射等]
- [配色建议]
- [布局方向]
```

### 第二页：诠释与过渡

```markdown
## 第二页：诠释与过渡

### 主题诠释
**核心问题**：[展厅要回答的终极问题]

**核心答案**：[展厅给出的答案，但不是直接告诉，而是让参观者自己发现]

### 叙事逻辑
**叙事结构**：[如：六幕式情感旅程、三段式叙事等]

**情感曲线**：
- 起点：[初始情感状态]
- 转折：[关键转折点]
- 高潮：[情感高潮]
- 终点：[最终情感目标]

### 空间规划策略
**空间布局**：
- [区域1]：[面积] - [功能] - [情感目标]
- [区域2]：[面积] - [功能] - [情感目标]
- [区域3]：[面积] - [功能] - [情感目标]

**动线设计**：[参观动线的设计思路]

### 视觉元素建议
- [过渡页的视觉设计建议]
- [如何从主题过渡到具体大纲]
```

### 第三页：详细的展陈大纲（要点版）

```markdown
## 第三页：详细的展陈大纲（要点版）

### 一、[区域名称]（[面积]）
**情感目标**：[这个区域要达到的情感目标]

**核心内容**：
1. **[模块1名称]**
   - 展示形式：[图文/互动/装置/体验]
   - 核心信息：[3-5个要点]
   - 叙事功能：[在整体叙事中的作用]

2. **[模块2名称]**
   - 展示形式：[图文/互动/装置/体验]
   - 核心信息：[3-5个要点]
   - 叙事功能：[在整体叙事中的作用]

### 二、[区域名称]（[面积]）
[同上结构]

### 三、[区域名称]（[面积]）
[同上结构]

### 展陈大纲总结
**总览**：
- 总空间：[总面积]
- 区域数量：[X个区域]
- 核心模块：[X个核心展示模块]
- 体验场景：[X个互动体验场景]

**叙事主线**：[一句话概括整个展陈的叙事主线]
```

## 生成原则

### 1. 结构清晰
- **第一页聚焦主题**：只展示展陈主题和核心理念，不涉及具体内容
- **第二页承上启下**：诠释主题，说明叙事逻辑，规划空间
- **第三页详细展开**：按区域组织，每个区域包含模块、内容、形式

### 2. 要点突出
- **每页不超过5个核心要点**：避免信息过载
- **用层级结构组织**：主标题→副标题→要点→细节
- **关键信息加粗**：突出核心概念和重要数据

### 3. 便于执行
- **展示形式明确**：每个模块都要说明展示形式（图文/互动/装置/体验）
- **叙事功能清晰**：说明每个模块在整体叙事中的作用
- **空间信息完整**：每个区域都要标注面积和功能定位

### 4. 参考样本特征
基于优秀展陈大纲样本（如某消费金融展厅展陈大纲（已脱敏））：
- **叙事驱动**：不是"物料清单"，而是"故事剧本"
- **情感曲线**：考虑参观者的情感变化
- **模块化设计**：每个模块独立且可组合
- **体验化呈现**：强调互动体验，而非静态展示

## 工作流程

### 步骤1：提取展陈主题
1. 从策划方案中提取展陈主题（主标题、副标题）
2. 提取主题的核心解读（空间维度、时间维度、统一理念）
3. 提取核心理念（如棱镜理念、液态叙事等）

### 步骤2：分析叙事逻辑
1. 识别策划方案中的叙事结构（如六幕式、三段式）
2. 提取情感曲线（起点→转折→高潮→终点）
3. 理解核心问题和核心答案

### 步骤3：规划空间布局
1. 从策划方案中提取空间规划信息
2. 识别各个区域的功能定位和情感目标
3. 理解动线设计思路

### 步骤4：组织展陈大纲
1. 按区域组织内容
2. 每个区域列出核心模块
3. 每个模块说明：展示形式、核心信息、叙事功能

### 步骤5：优化与检查
1. **结构检查**：确保三页内容层次清晰，逻辑递进
2. **要点检查**：确保每页核心要点不超过5个
3. **完整性检查**：确保空间信息、展示形式、叙事功能都完整

## 示例

### 输入（策划方案片段）
```markdown
## 核心策划理念：无界触达

> **展陈主题**：无界触达
> **副标题**：瞬间即永恒 此刻即未来

**"无界触达"**：用AI重新定义金融服务的时空边界，让金融服务触达每一个人。

**空间维度：无界触达**
- **无界**：AI重构成本，消除了金融服务的边界
- **触达**：让金融服务触达每一个人

**时间维度：瞬间即永恒 此刻即未来**
- **瞬间**：每一次"即刻"的响应
- **永恒**：持续的价值创造
```

### 输出（第一页：展陈主题）
```markdown
## 第一页：展陈主题

### 主标题
无界触达

### 副标题
瞬间即永恒 此刻即未来

### 主题核心解读
- **一句话概括**：用AI重新定义金融服务的时空边界，让金融服务触达每一个人
- **空间维度**：AI重构成本，消除边界（成本/距离/门槛），让服务触达
- **时间维度**：每一次"即刻"的响应（瞬间）汇聚成持续的价值创造（永恒）
- **统一理念**：AI重构成本→消除边界→让金融服务触达每一个人

### 核心理念
棱镜理念：某消费金融公司如白光，展厅如棱镜，为每位参观者折射专属光谱体验。

### 视觉元素建议
- 主视觉：数十米数据星河装置（每一次闪烁代表"此刻"的交易）
- 配色：科技蓝+金色渐变
- 布局：主题居中，副标题下方，理念解读左右分布
```

### 输出（第二页：诠释与过渡）
```markdown
## 第二页：诠释与过渡

### 主题诠释
**核心问题**：在这个充满不确定性的时代，普惠金融的未来是什么样的？

**核心答案**：某消费金融公司 = 用AI重新定义金融服务成本结构的普惠金融基座
（但我们不是"告诉"参观者这个答案，而是让他们自己"发现"）

### 叙事逻辑
**叙事结构**：六幕式情感旅程

**情感曲线**：
- 起点：疲惫、防备、略带期待（入口区）
- 转折1：从好奇转向信任（过渡区）
- 转折2：从信任转向理解（业务区）
- 高潮：从理解转向震撼（科技区）
- 转折3：从震撼转向确信（体验区）
- 终点：从确信转向共鸣（出口区）

### 空间规划策略
**空间布局**：
- 入口区（50㎡）- 唤醒好奇 - 情感目标：打破心防
- 过渡区（30㎡）- 建立信任 - 情感目标：从好奇转向信任
- 业务区（80㎡）- 传递认知 - 情感目标：从信任转向理解
- 科技区（120㎡）- 制造震撼 - 情感目标：从理解转向震撼
- 体验区（80㎡）- 亲身验证 - 情感目标：从震撼转向确信
- 出口区（65㎡）- 情感共鸣 - 情感目标：从确信转向共鸣

**动线设计**：单向流线，从入口到出口，情感逐步递进

### 视觉元素建议
- 过渡页：情感曲线图（从起点到终点的情感变化）
- 空间布局示意图（六个区域的分布）
- 配色：延续主题页的科技蓝+金色
```

### 输出（第三页：详细的展陈大纲要点版）
```markdown
## 第三页：详细的展陈大纲（要点版）

### 一、入口区（50㎡）
**情感目标**：打破心防，建立好奇

**核心内容**：
1. **数十米数据星河装置**
   - 展示形式：光纤阵列实时闪烁装置
   - 核心信息：
     - 每一次闪烁代表一笔真实发生的交易/服务
     - 超高挑空区，视觉震撼
     - 无需文字说明，用视觉制造"WOW"时刻
   - 叙事功能：建立规模感，"这家公司在做很大的事"

2. **时光隧道**
   - 展示形式：时间轴+动态光效
   - 核心信息：
     - 左侧：企业发展大事记
     - 右侧：中国普惠金融发展大事记（对照）
     - 随参观者行进，时间轴逐渐点亮
   - 叙事功能：建立时间感，"这家公司有历史沉淀"

### 二、过渡区（30㎡）
**情感目标**：从好奇转向信任

**核心内容**：
1. **荣誉星辰**
   - 展示形式：触控星座墙
   - 核心信息：
     - 以"星座"形式呈现各类荣誉
     - 核心荣誉（国家企业技术中心等）用更大的"星"标识
     - 触摸任意"星"，弹出详情
   - 叙事功能：权威背书，"这家公司被认可"

2. **见证者之墙**
   - 展示形式：时间轴互动墙
   - 核心信息：
     - 展示历年领导莅临场景
     - 时间轴滚动：2015年→2025年
     - 点击可展开详情
   - 叙事功能：历史见证，"这家公司有重量级见证者"

### 三、业务区（80㎡）
**情感目标**：从信任转向理解

**核心内容**：
1. **一个用户的旅程**
   - 展示形式：大屏动画+互动
   - 核心信息：
     - 虚拟人物"小李"的故事自动播放
     - 展示：智能获客→智能风控→智能客服→智能资管→资产回收
     - 每个环节可点击深入
   - 叙事功能：业务逻辑翻译，"原来他们是这样服务用户的"

2. **数据心跳**
   - 展示形式：实时数据大屏
   - 核心信息：
     - 实时展示：累计服务用户数、累计放款金额、今日审批数等
     - 数据"跳动"，每秒更新
     - 配合声效：每隔几秒，听到"心跳"
   - 叙事功能：规模感知，"这个规模太惊人了"

### 四、科技区（120㎡）
**情感目标**：从理解转向震撼

**核心内容**：
1. **企业自研大模型装置**
   - 展示形式：物理装置+互动演示
   - 核心信息：
     - 发光的球体或多面体，内部有动态光效
     - 周围环绕6个能力模块
     - 每个模块配合演示视频/交互Demo
   - 叙事功能：核心技术展示，"这个AI太强了"

2. **能力图谱**
   - 展示形式：触控矩阵
   - 核心信息：
     - 四大模块：AI赋能的普惠金融旅程、多模态融合的金融智能核心引擎、可信金融AI基座与安全护航体系、智能驱动的高效创新双螺旋
     - 点击进入二级页面
   - 叙事功能：技术全景，"技术覆盖面太广了"

### 五、体验区（80㎡）
**情感目标**：从震撼转向确信

**核心内容**：
1. **AI面试体验**
   - 展示形式：交互体验
   - 核心信息：
     - AI面试官（数字人形态）提出问题
     - 参观者回答，AI实时分析
     - 生成面试评价报告
   - 叙事功能：技术验证，"这AI真能面试！"

2. **人脸识别安全体验**
   - 展示形式：交互体验
   - 核心信息：
     - 系统完成人脸识别、活体检测
     - 尝试用照片/视频"欺骗"系统（失败）
   - 叙事功能：安全感知，"这系统骗不了"

3. **数字人克隆体验**
   - 展示形式：交互体验
   - 核心信息：
     - 参观者录制30秒视频
     - AI实时生成"数字人分身"
   - 叙事功能：WOW时刻，"我也有数字人了！"

### 六、出口区（65㎡）
**情感目标**：从确信转向共鸣

**核心内容**：
1. **温度故事**
   - 展示形式：故事墙+互动
   - 核心信息：
     - 企业公益基金会介绍
     - 6大公益项目：就业帮、养老机器人、智慧养鸡等
     - 每个项目都有具体的受益者故事
   - 叙事功能：价值观传递，"这家公司有温度"

2. **数智足迹**
   - 展示形式：离馆报告系统
   - 核心信息：
     - 扫码获取专属体验报告
     - 报告内容根据参观者的实际体验动态生成
     - 包含后续跟进的钩子（白皮书、报告等）
   - 叙事功能：体验延续，"带走了专属记忆"

### 展陈大纲总结
**总览**：
- 总空间：425㎡
- 区域数量：6个区域
- 核心模块：15个核心展示模块
- 体验场景：4个互动体验场景

**叙事主线**：从"仰望星河"到"带走星光"——一个用户如何在45分钟内，从陌生走向共鸣。
```

## 注意事项

1. **避免信息过载**：每页PPT的核心要点不超过5个
2. **保持逻辑递进**：三页内容要有清晰的逻辑关系（主题→诠释→大纲）
3. **强调叙事功能**：每个模块都要说明在整体叙事中的作用
4. **空间信息完整**：每个区域都要标注面积和功能定位
5. **展示形式明确**：每个模块都要说明展示形式（图文/互动/装置/体验）
6. **参考样本特征**：学习优秀展陈大纲的叙事驱动、情感曲线、模块化设计等特点
