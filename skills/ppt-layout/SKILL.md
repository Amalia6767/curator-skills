# PPT排版指南生成

## 目标

将策划方案（Markdown格式）转化为平面设计师可直接使用的PPT排版指南，确保：
- **按页码组织**：每页独立，便于设计师逐页执行
- **层级清晰**：标题、副标题、正文、强调文本的层级关系明确
- **内容精炼**：去除冗余，保留核心信息
- **创意提示**：提供视觉化建议，激发设计灵感

## 输入

- 策划方案（Markdown格式，通常来自 `proposal-design` skill）
- 可选：参考样本PPT（用于学习排版风格）

## 输出格式

每页PPT排版指南包含以下结构：

```markdown
## 第X页：[页面标题]

### 页面定位
[一句话说明这页在整个方案中的位置和作用]

### 标题层级
- **主标题**：[标题文字]
- **副标题**（可选）：[副标题文字]
- **章节编号**（如适用）：[编号]

### 内容层级
1. **一级内容**：[核心信息]
   - 二级内容：[支撑信息]
   - 二级内容：[支撑信息]
2. **一级内容**：[核心信息]
   - 二级内容：[支撑信息]

### 视觉元素建议
- [视觉化建议，如：图表类型、图标、配色、布局方向]

### 一句话总结
[用一句话概括这页要传达的核心信息]
```

## 排版原则

### 1. 信息密度控制
- **封面/封底**：极简，仅核心信息
- **章节页**：标题+一句话理念
- **内容页**：3-5个要点，每点不超过2行
- **数据页**：图表为主，文字为辅

### 2. 视觉层级
- **主标题**：最大字号，视觉焦点
- **副标题**：次大字号，补充说明
- **一级内容**：加粗，清晰可读
- **二级内容**：常规字号，支撑细节

### 3. 创意表达
- **理念页**：用视觉隐喻（如棱镜、光谱、液态）
- **数据页**：用动态图表、信息图
- **流程页**：用时间轴、流程图
- **对比页**：用表格、矩阵图

### 4. 参考样本风格
基于优秀样本（如某智慧能源集团、新加坡陆军博物馆）：
- **简洁现代**：留白充足，信息不拥挤
- **结构化**：用表格、清单、流程图组织信息
- **视觉化**：少文字，多图表、图标、示意图
- **一致性**：统一的配色、字体、间距

## 工作流程

### 步骤1：分析策划方案结构
1. 识别主要章节（通常对应PPT的章节页）
2. 识别每个章节下的子内容（通常对应内容页）
3. 识别需要视觉化的内容（数据、流程、对比）

### 步骤2：规划PPT页面结构
1. **封面页**：项目名称、副标题、日期
2. **目录页**：主要章节导航
3. **章节页**：每个主要章节的过渡页
4. **内容页**：具体内容展示
5. **封底页**：联系方式、感谢

### 步骤3：逐页生成排版指南
对每一页：
1. 提取核心信息
2. 确定标题层级
3. 组织内容层级
4. 提供视觉化建议
5. 写一句话总结

### 步骤4：优化与检查
1. **一致性检查**：确保层级关系统一
2. **精简检查**：删除冗余信息
3. **创意检查**：确保每页都有视觉化建议

## 示例

### 输入（策划方案片段）
```markdown
## 核心策划理念：棱镜 3.0

某消费金融公司就像一束白光，蕴含着丰富的"光谱"。
这个展厅就是一面精密的"棱镜"。
当不同的参观者到来时，它能精准地折射出最适合对方观看的"光谱"。
```

### 输出（PPT排版指南）
```markdown
## 第3页：核心策划理念——棱镜3.0

### 页面定位
核心理念页，奠定整个方案的哲学基础

### 标题层级
- **主标题**：核心策划理念
- **副标题**：棱镜 3.0 —— 液态叙事的数智展厅

### 内容层级
1. **核心隐喻**
   - 某消费金融公司 = 白光（蕴含丰富光谱）
   - 展厅 = 棱镜（精准折射）
   - 参观者 = 专属光谱（千人千面）

2. **理念升级**
   - 1.0：固态展示（内容固定）
   - 2.0：半固态（可变形但缓慢）
   - 3.0：液态叙事（千人千面动态适配）

### 视觉元素建议
- 主视觉：棱镜折射示意图（一束白光进入，多色光谱射出）
- 配色：白色→渐变色光谱（蓝、金、红等）
- 布局：左侧文字，右侧示意图
- 动画建议：光线进入→折射→光谱展开

### 一句话总结
展厅如棱镜，将某消费金融公司的白光折射为每位参观者的专属光谱体验。
```

## 特殊页面处理

### 封面页
- **标题**：项目全称（大字号）
- **副标题**：设计施工一体化工程（中字号）
- **底部**：日期、版本号
- **视觉**：核心视觉元素（如数据星河、棱镜）

### 目录页
- **标题**：目录 / Contents
- **内容**：主要章节列表（3-5项）
- **视觉**：简洁的导航图标或数字编号

### 章节过渡页
- **标题**：章节名称（大字号）
- **副标题**：一句话理念（中字号）
- **视觉**：章节主题相关的抽象图形

### 数据页
- **标题**：数据主题
- **内容**：图表为主，文字为辅
- **视觉**：动态图表、信息图、数据可视化

### 对比页
- **标题**：对比主题
- **内容**：表格或矩阵
- **视觉**：清晰的表格设计，用颜色区分维度

## 注意事项

1. **避免信息过载**：每页不超过5个要点
2. **保持视觉一致性**：统一的配色、字体、间距
3. **强调核心信息**：用加粗、颜色、大小突出关键点
4. **提供创意方向**：不仅说"放什么"，还要说"怎么放"
5. **考虑动画效果**：对复杂内容建议动画呈现方式
