# PPT排版指南生成 Skill

本 skill 用于将策划方案（Markdown格式）转化为平面设计师可直接使用的PPT排版指南。按页码组织，每页包含标题层级、内容层级、视觉元素建议和一句话总结，确保理念清晰、不繁冗、有创意。

## 文件结构

```
ppt-layout/
├── SKILL.md          # 核心技能说明与工作流程
└── README.md         # 本文件
```

## 输入

### 必需输入
- **策划方案**（Markdown格式）：通常来自 `proposal-design` skill 的输出
  - 例如：`投标方案_策划篇.md`、`展陈大纲_叙事版.md`

### 可选输入
- **参考样本PPT**：用于学习排版风格
  - 例如：某智慧能源集团、新加坡陆军博物馆等优秀样本
  - 位置：`投标方案学习样本/` 目录

## 输出

生成 `PPT排版指南.md` 文件，包含：

1. **封面页**：项目名称、副标题、日期
2. **目录页**：主要章节导航
3. **章节过渡页**：每个主要章节的过渡页
4. **内容页**：逐页排版指南，每页包含：
   - 页面定位
   - 标题层级（主标题、副标题）
   - 内容层级（一级、二级内容）
   - 视觉元素建议（图表类型、配色、布局）
   - 一句话总结

## 使用场景

- **策划 → 设计交接**：策划完成方案后，需要平面设计师制作PPT
- **标准化排版**：确保PPT风格统一、信息层级清晰
- **创意指导**：为设计师提供视觉化建议，激发设计灵感

## 工作流程

1. **分析策划方案结构**：识别章节、子内容、需要视觉化的部分
2. **规划PPT页面结构**：封面、目录、章节页、内容页、封底
3. **逐页生成排版指南**：提取核心信息，组织层级，提供视觉建议
4. **优化与检查**：一致性、精简性、创意性检查

## 排版原则

- **信息密度控制**：封面极简，内容页3-5个要点
- **视觉层级清晰**：主标题>副标题>一级内容>二级内容
- **创意表达**：理念页用视觉隐喻，数据页用图表
- **参考样本风格**：简洁现代、结构化、视觉化、一致性

## 与 proposal-design 的关系

- `proposal-design`：生成策划方案（Markdown）
- `ppt-layout`：将策划方案转化为PPT排版指南
- 两者配合使用，完成从需求分析到设计交付的完整流程
